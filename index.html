HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • An element is a type of content on a webpage.
  • In semantic HTML elements, the name of the element describes its content and the role that it plays on a webpage.
  • The body element is an example of a semantic HTML element.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A selector defines the element or attributes to which the rules, or declarations apply.
  • You can combine two selectors into a single CSS rule with a comma, allowing the declarations to affect both selectors.

Git

  • git status: checks what branch I'm currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add: ad a file as it looks now next to your next commit
  • git pull: copies changes from a remote repo directly into your working directory

JavaScript

  • JavaScript is a high-level dynamic programming language that allows users to interact with websites. Using JavaScript, you can power your websites with effects, interactivity, data communication, and much more!
  • In JavaScript, to declare a variable we use the "var" keyword, after the "var" keyword, we give the variable a name.
  • All strings are surrounded by quotation marks.
  • At the most basic level a function is a set of instructions that tells a computer how to do a certain task.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A functions name is followed by parentheses
  • An if statement is a way to introduce decision making into our code
  • An array is a single variable that is used to hold a group of data. Arrays are typically used to hold data that is related to them in some way.
  • A for loop uses the predictable pattern of indces to perform a task on all the items in an array by allowing a single code block to be executed over and over.
