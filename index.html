HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • An element is a type of content on a webpage.
  • In semantic HTML elements, the name of the element describes its content and the role that it plays on a webpage.
  • The body element is an example of a semantic HTML element.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A selector defines the element or attributes to which the rules, or declarations apply.
  • You can combine two selectors into a single CSS rule with a comma, allowing the declarations to affect both selectors.

Git

  • git status: checks what branch I'm currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add: ad a file as it looks now next to your next commit
  • git pull: copies changes from a remote repo directly into your working directory
